## 📈 EasyInvest
Investing, Finally Simple.

EasyInvest is a concept investing platform designed to make stock market investing more approachable, educational, and user-friendly for beginners.

Inspired by Yahoo Finance, Robinhood, and Schwab, EasyInvest explores how real-time market data, portfolio management, financial education, and AI-powered guidance can be combined into a single intuitive experience.

## 🎯 Project Overview

Many investing platforms fall into one of two extremes:

- 📈 Powerful platforms provide extensive financial data but can overwhelm beginners.

- 📱 Simple investing apps make trading easy but often lack educational support and deeper insights.

EasyInvest was created to bridge this gap by helping users learn while they invest through thoughtful UX design, simplified financial information, and AI-assisted guidance.

## 💡 Solution

EasyInvest reimagines investing through:

- 📈 Beginner-friendly dashboards
- 💼 Portfolio tracking
- 🤖 AI-powered investing assistant
- 📚 Built-in financial education
- 📊 Simplified data visualizations
- 🎨 Clean and intuitive user experience

The goal is to help users understand investments rather than simply trade them.

## 🚀 Getting Started
- Low-Fidelity MVP

    - Open directly in your browser:
    - easyinvest-live.html
    - No installation required.

- Functional Prototype
    - Install dependencies:
        - npm install
        - Run development server:
            - npm run dev
        - Run backend:
            - npm run server
        - Run both:
            - npm run dev:full
